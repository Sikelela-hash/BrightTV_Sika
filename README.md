# BrightTV_Sika
For CEO

In this project, the BrightTV Analysis was conducted to evaluate platform performance over a six-month period, using user viewership and engagement data across all provinces and channels.
This analysis provides a comprehensive view of user behavior, content consumption patterns, demographic insights (race, gender, province), and viewing duration, enabling data driven decision making at an executive level.
The goal is to understand key drivers of viewership, identify growth opportunities, optimize content strategy, and improve overall platform performance and user experience.

--Objectives--

To analyze viewership performance across all channels over six months
To identify usage trends and peak viewing periods
To determine most watched channels and top performing content
To evaluate user behavior by race, gender, province, and viewing duration
To identify low consumption periods and improvement opportunities
To provide actionable insights for user growth and engagement optimization

--Tools Used--

Miro: for planning, brainstorming, and structuring the analysis
Databricks (SQL): for data extraction, transformation, and analysis of user activity
Microsoft Excel: for data cleaning, pivot tables, trend analysis, and graphs
Canva: for visualizing insights and presenting to CEO


--Project Taks--

Extracted and consolidated 6 months of BrightTV user activity data
Performed SQL analysis to calculate:
Total viewership
Watch time (duration)
Channel performance
User segmentation (race, gender, province)
Analyzed time-based viewing patterns:
Morning
Afternoon
Evening (peak consumption)
Identified:
Most watched channels and content categories
High-engagement user segments
Provincial performance trends
Built dashboards and visual reports to highlight key insights
Compiled findings into a CEO-level presentation
